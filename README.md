# Cybersecurity-Risk-Assessment
Cybersecurity Risk Assessment using Data Analytics and Machine Learning
This project analyzes global cybersecurity incidents to understand cyber risks across countries and industries using data analytics, statistical analysis, machine learning, and a custom Cybersecurity Risk Index.
## Objectives
- Explore cybersecurity incident trends
- Compare countries and industries
- Perform statistical hypothesis testing
- Build predictive machine learning models
- Develop a Cybersecurity Risk Index
- Provide policy recommendations
## Dataset
Source:
Kaggle Global Cybersecurity Threats Dataset
